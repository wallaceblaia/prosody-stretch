# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-01-20

- [Added] Initial release
- [Added] Core library with ProsodyStretcher API
- [Added] WSOLA time-stretching via pytsmod
- [Added] Pause manipulation strategy
- [Added] Vowel extension strategy
- [Added] CLI with adjust, match, info, convert commands
- [Added] REST API with FastAPI
- [Added] Support for WAV, MP3, FLAC, OGG input formats
- [Added] Sample rate and channel conversion options
